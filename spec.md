# OpenClaw智能铁道机车探伤机器人系统 PPT 规格文档

## 1. 项目概述

### 项目名称
OpenClaw智能铁道机车探伤机器人系统 - reveal.js 交互式演示

### 项目目标
基于原始技术文档，使用 reveal.js + AI生成媒体资源（视频/图片），制作一款高信息密度的交互式WebPPT，完整呈现文档全部内容，一字不省略。

### 核心要求
- 所有文档内容完整呈现，不删减、不改写
- 每页包含独立的AI生成背景图和AI生成配图，两者内容不同但主题相关
- 封面页配备AI生成的无限循环视频背景
- 封面页支持：首次打开自动播放、左键返回封面时重新播放

## 2. 技术栈

| 组件 | 技术选型 | 用途 |
|------|---------|------|
| 演示框架 | reveal.js 5.x | HTML幻灯片框架 |
| 动画插件 | reveal.js Appearance | 元素逐个出现动画 |
| 动画库 | Anime.js 3.x | 自定义复杂动画效果 |
| AI图像生成 | SiliconFlow Kolors | 每页背景图和配图生成 |
| AI视频生成 | SiliconFlow Wan2.2-T2V | 封面无限循环视频生成 |
| 布局技术 | CSS Grid + Flexbox | 多列紧凑布局 |
| 字体 | 思源黑体 / 等宽字体 | 高信息密度排版 |

## 4. 视觉设计规范

### 4.1 背景图规范

| 属性 | 规格 |
|------|------|
| 尺寸 | 1920×1080 (16:9) |
| 透明度 | 20-25% |
| 位置 | 全屏覆盖，data-background |
| 风格 | 抽象科技纹理、粒子光线效果，蓝/青色调 |
| 数量 | 44张（每页一张独立的背景图） |

### 4.2 配图规范

| 属性 | 规格 |
|------|------|
| 尺寸 | 页面宽度的30-40% |
| 透明度 | 35-45% |
| 位置 | 页面右侧或角落，可浮动 |
| 风格 | 内容相关内容图，与背景图风格统一但内容不同 |
| 数量 | 44张（每页一张独立的配图） |

### 4.3 封面视频规范

| 属性 | 规格 |
|------|------|
| 时长 | 5秒（AI生成） |
| 分辨率 | 1024×576 或 1920×1080 |
| 格式 | MP4 |
| 效果 | 无限循环，muted自动播放 |
| 提示词 | 未来机器人+铁路场景，科幻风格，"loopable" |

### 4.4 字体与排版

| 元素 | 字号 | 行高 | 颜色 |
|------|------|------|------|
| 页面标题 | 28-32px | 1.3 | #FFFFFF |
| 二级标题 | 20-24px | 1.3 | #4FC3F7 |
| 正文 | 14-16px | 1.4 | #E0E0E0 |
| 列表项 | 12-14px | 1.3 | #BDBDBD |
| 页码 | 12px | - | #666666 |

## 5. AI生成资源清单

### 5.1 视频（1个）
- 封面无限循环视频：未来机器人铁路检测主题

### 5.2 背景图（44张）
- 每页一张独立背景图
- 统一风格：科技蓝调抽象纹理
- 主题与页面内容相关

### 5.3 配图（44张）
- 每页一张独立配图
- 尺寸小于背景图
- 主题与背景图不同但相关
- 内容直接关联页面文字

---

## 6. 交互设计

### 6.1 导航
- 左右箭头切换页面
- 支持键盘导航（←/→）
- 支持触摸滑动

### 6.2 封面视频行为
- 网页加载完成自动播放
- 从任何页面返回封面时重新播放
- 离开封面时继续循环播放（隐藏状态）

### 6.3 动画效果
- 页面切换：普通滑动过渡
- 内容出现：逐句/逐段淡入动画（Appearance插件）
- 配图：轻微浮动动画
