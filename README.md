# OpenClaw智能铁道机车探伤机器人系统 Web PPT

这是一个基于reveal.js的Web演示文稿，介绍OpenClaw智能铁道机车探伤机器人系统。

## 在线访问

访问地址：https://wya20.github.io/openclaw-robot-ppt/

## 技术栈

- reveal.js 5.x - HTML演示文稿框架
- reveal.js Appearance - 动画插件
- Anime.js - 动画引擎

## 本地运行

```bash
# 使用Python启动本地服务器
python -m http.server 8080
```

然后访问 http://localhost:8080
